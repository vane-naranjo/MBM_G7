# Proyecto: Comparación de la microbiota intestinal en individuos obesos y normopeso mediante secuenciación 16S usando Galaxy

## Objetivo
Analizar diferencias en la composición de la microbiota intestinal entre individuos obesos y normopeso mediante datos de secuenciación 16S utilizando la plataforma Galaxy.

## Dataset
Se utilizarán datasets públicos de microbiota intestinal obtenidos desde bases de datos como SRA/ENA relacionados con obesidad y secuenciación 16S.

## Flujo de trabajo
1. Obtención de datasets públicos
2. Control de calidad con FastQC
3. Filtrado y limpieza de secuencias
4. Clasificación taxonómica
5. Visualización de microbiota
6. Interpretación biológica

## Resultados esperados
Identificar diferencias en diversidad y abundancia bacteriana entre individuos obesos y normopeso.
